# Comanda-Digital - Backend

Este é o backend do sistema **Comanda-Digital**, desenvolvido com **Spring Boot** e integrado ao banco de dados **MySQL**. O backend também serve os arquivos do frontend Angular, permitindo que a aplicação seja executada a partir de um único servidor.

---

## Pré-requisitos

1. **Java 17 ou superior**:
   - Verifique se está instalado com o comando:
    
     java -version
     

2. **MySQL**:
   - Certifique-se de que o MySQL está instalado e rodando.
   - O banco de dados deve ser configurado com as credenciais abaixo:
     - **Nome do Banco**: `comanda_digital`
     - **Usuário**: `root`
     - **Senha**: `iBeus2024`

---

## Configuração do Banco de Dados

1. **Crie o Banco de Dados**:
   Abra o terminal do MySQL e execute o seguinte comando:
   
   CREATE DATABASE comanda_digital;

####   PARA RODAR O PROJETO  ####
   
   Vá ate a pasta cd target

   digite java -jar Comanda-Digital-0.0.1-SNAPSHOT.jar

   logo após digite http://localhost:8080 no navegador 
   
   e o projeto estará rodando.  :D
